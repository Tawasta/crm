==========================
d.m.Y dateformat for en_US
==========================

* Sets the date format of the en_US locale to be %d.%m.%Y (e.g. 31.12.2018)
* Intended for situations where e.g. all dates on prints should use the
  Finnish date format, regardless of the partner's language.
